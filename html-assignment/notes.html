5 HTML tags with descriptions

  • <h1> - Heading tag that defines the largest and most important heading on a page
  • <p> - Paragraph tag that defines a block of text
  • <a> - Anchor tag that creates a hyperlink to another web page or resource
  • <img> - Image tag that embeds an image into an HTML document
  • <div> - Division tag that defines a container for HTML elements and is often used for layout purposes
